# Ocean Cleaning — Design System

Бренд и дизайн-система для **Ocean Cleaning** — профессионального клининга для
ресторанов, кафе и заведений HoReCa в Алматы. Весь продуктовый текст — на русском.

> Это дизайн-система, создаваемая «с нуля» по брифу заказчика. Готового кодовой
> базы или Figma-файла не было — все токены, компоненты и UI kit собраны по
> текстовому описанию бренда.

---

## Контекст продукта

- **Продукт:** одностраничный лендинг (single-page landing).
- **Аудитория:** управляющие и владельцы ресторанов и кафе, которым нужна
  глубокая чистка мебели и ковролина, устранение запахов и дезинфекция.
- **Задача:** показать услуги, объяснить «почему мы», дать контакты и кнопку звонка.
- **Тон:** профессиональный, чистый, доверительный, без агрессивного маркетинга.

### Источники
Кодовой базы / Figma не предоставлено. Единственный источник — текстовый бриф
заказчика (структура лендинга, цвета, описание логотипа и услуг). Реальные фото
интерьеров и официальный файл логотипа **нужно получить у заказчика** — сейчас
используются заглушки (см. CAVEATS внизу).

## CONTENT FUNDAMENTALS (как писать тексты)

- **Язык:** русский. Всегда.
- **Обращение:** на «вы» к заведению/гостю — «для вашего ресторана»,
  «в удобное для вас время», «чувствуют ваши гости». Без панибратства.
- **Голос компании:** «мы» — «Удаляем пятна…», «Уничтожаем бактерии…»,
  «Возвращаем мебели первозданный вид». Глаголы в настоящем времени, 1-е лицо мн.ч.
- **Регистр:** заголовок hero — ПРОПИСНЫМИ («ЧИСТОТА И СВЕЖЕСТЬ ДЛЯ ВАШЕГО
  РЕСТОРАНА»). Заголовки секций — обычным регистром. Надзаголовки (eyebrow) —
  капсом с разрядкой («НАШИ УСЛУГИ»).
- **Длина:** короткие ёмкие фразы. Описание услуги — 1–2 предложения.
- **Пунктуация-акценты:** допускается восклицание в дружелюбных «бейджах»
  («Специальные условия для постоянных клиентов!»).
- **Эмодзи:** НЕ используются. Никогда.
- **Чего избегаем:** счётчиков и цифр статистики («10 000 клиентов»),
  дублирующихся кнопок призыва, превосходных степеней без основания.

Примеры тона:
> «Профессиональные клининговые услуги для ресторанов, кафе и заведений HoReCa.»
> «Эффективное устранение неприятных запахов: кухни, табака, сырости, животных и других.»
> «Чистота, которую видно. Свежесть, которую чувствуют ваши гости!»

## VISUAL FOUNDATIONS

- **Цвет.** Основной — насыщенный океанский / royal blue `--ocean-600 #1E5FD1`.
  Фон страницы — белый `--white`. Нижняя контактная панель и футер — тёмно-синий
  navy (`--ocean-900 #0C2A54` / `--ocean-950`). Лёгкий «свежий» аква-тон
  (`--aqua-*`) для мягких подложек. Серая шкала — холодная (синеватая).
- **Типографика.** Дисплейный шрифт — **Onest** (заголовки, hero, цифры),
  начертания 700/800, плотный трекинг. Текстовый — **Manrope** (абзацы, подписи),
  400–600. Оба с кириллицей. Hero-заголовок — капсом.
- **Фоны.** Плоские заливки (белый / soft-gray / navy). Без фотографических
  фонов-героев, без шумов и текстур. Допустим один диагональный сине-голубой
  градиент только на фото-заглушках.
- **Разделители секций.** Фирменная плавная **волна** (`WaveDivider`) на стыке
  секций — наследует цвет следующей секции. Это ключевой мотив бренда (вода/океан).
- **Бейджи услуг.** Круглые синие бейджи (`--color-primary`) с белой контурной
  иконкой и мягкой синей тенью `--shadow-blue`.
- **Пунктирные акценты.** Дружелюбные заметки в пунктирной обводке
  (`2px dashed --ocean-300`) — круг поверх hero-фото (иконка листа) и
  скруглённый блок-оффер (иконка подарка).
- **Скругления.** Кнопки — pill (`--radius-pill`). Карточки/блоки — `md/lg`
  (16–24px). Бейджи — полный круг.
- **Тени.** Мягкие, синеватые, низкие (`--shadow-sm/md/lg`), плюс отдельная
  «синяя» тень под основными CTA и бейджами (`--shadow-blue`).
- **Границы.** Тонкие (`1.5px`) `--border-subtle`. Пунктир — `2px`.
- **Состояния.** Hover — лёгкое затемнение (`brightness .94`); press — сдвиг на
  1px вниз. Фокус — синее кольцо `--ring`.
- **Анимации.** Нет. Бренд статичный — без счётчиков, без появлений, без бунса.
  Допустимы только микропереходы hover/press на кнопках.
- **Прозрачность/блюр.** Минимально: лёгкий backdrop-blur у круглого hero-бейджа.
- **Тон изображений.** Тёплые, светлые, чистые интерьеры; ровный дневной свет.
  Без ч/б, без зерна, без тяжёлых фильтров.
- **Лейаут.** Контейнер `max 1180px`, секции с крупными вертикальными отступами
  (`--space-9/10`), сетки 4 колонки (услуги) и 2 колонки (преимущества).

## ICONOGRAPHY

- **Система:** только аккуратные **контурные (line) иконки**. Используется
  **[Lucide](https://lucide.dev)** (UMD CDN, pinned `0.460.0`), stroke-width `1.75`.
  Эмодзи и юникод-символы как иконки — **не используются**.
- **Подключение:** на странице — `<script src="https://unpkg.com/lucide@0.460.0/dist/umd/lucide.min.js"></script>`;
  в React — компонент `Icon` (обёртка над Lucide, см. `components/core/Icon.jsx`).
- **Карта иконок бренда:**
  - `sofa` — химчистка мягкой мебели
  - `layers` — химчистка ковролина
  - `spray-can` — сухой туман
  - `shield-check` — дезинфекция
  - `check` — пункты «почему выбирают нас»
  - `leaf` — hero-бейдж «чистота/свежесть»
  - `gift` — оффер для постоянных клиентов
  - `phone` / `instagram` / `map-pin` — контактная панель
- **Логотип:** круглая синяя волна с силуэтами чаек. Сейчас — **placeholder**
  (`assets/logo.svg` + компонент `Logo`). Заменить на официальный файл бренда.

> ⚠️ Lucide — **подстановка** под требование «контурные иконки» (готового
> айкон-сета бренд не предоставил). Если у бренда есть свой набор — заменить.

## Структура проекта (index / manifest)

```
styles.css              # точка входа: только @import-ы
tokens/
  fonts.css             # Onest + Manrope (Google Fonts CDN)
  colors.css            # океан-шкала, navy, нейтрали, семантика
  typography.css        # шрифты, шкала размеров, веса, трекинг
  spacing.css           # 8px-ритм, контейнер
  effects.css           # радиусы, границы, тени, focus ring
  base.css              # .oc-icon, .oc-page хелперы
components/
  core/      Button, Icon                         (+ core.card.html)
  brand/     Logo                                  (+ brand.card.html)
  marketing/ ServiceBadge, FeatureCheck,           (+ marketing.card.html,
             DashedCallout, ContactCard,             contact.card.html)
             WaveDivider, SectionHeading
guidelines/                # foundation specimen cards (Type/Colors/Spacing/Brand)
ui_kits/
  landing/   index.html + Sections.jsx            # полный лендинг
assets/
  logo.svg                # placeholder-логотип
SKILL.md                  # обёртка для Agent Skills / Claude Code
```

**Компоненты:** `Button`, `Icon`, `Logo`, `ServiceBadge`, `FeatureCheck`,
`DashedCallout`, `ContactCard`, `WaveDivider`, `SectionHeading`.

**UI kit:** `ui_kits/landing/` — пиксельная сборка лендинга из компонентов.
